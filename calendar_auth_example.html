To enable calendar-aware context and smart reminders, we need to connect your Google Calendar.

Quick Setup Instructions:

  1. Go to Google Cloud Console
  2. Create a new project or select existing
  3. Enable the Google Calendar API
  4. Create OAuth 2.0 credentials (Desktop type)
  5. Download the credentials JSON
  6. Save it as credentials.json in the ADHDo folder
  7. Run: python src/mcp_server/setup_calendar.py

Alternative: Use Service Account

For server-side access without browser auth, you can use a service account:

  1. Create a service account in Google Cloud Console
  2. Download the service account key JSON
  3. Share your calendar with the service account email
  4. Set GOOGLE_SERVICE_ACCOUNT_FILE environment variable
